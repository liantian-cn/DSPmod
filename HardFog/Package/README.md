# HardFog

HardFog adds a UXAssist configuration tab for Dark Fog cleanup tools and several merged convenience toggles. Cheating-oriented toggles are off by default unless noted.

## Features

- Clear Dark Fog on the current planet.
- Clear space Dark Fog in the current star system.
- Fill the galaxy with Dark Fog hives.
- Generate Dark Fog surface ruins on the current planet and build geothermal power stations on idle ruins.
- Dark Fog Pressure Reducer: slowly lowers non-active hive threat, off by default.
- Relay stations only dispatch to markers, off by default.
- Research Speed Multiplier: reduces research hash requirements to about 1/36, off by default.
- Ignore ground support requirement: treats missing ground or foundation support as buildable, including water placement, on by default.
- Better vein placement: relocates newly generated vein groups from each planet birth point into easier-to-plan equatorial longitude windows, on by default.
- Fighters fly farther: boosts mecha-owned fighters with 10x range, 10x damage, and invincibility, off by default.

## Configuration

Open the UXAssist panel with `Alt + BackQuote` and use the HardFog tab.

## Notes

- Better vein placement only affects future planet vein generation.
- Fighters fly farther only affects mecha-owned fighters; battle base fighters are excluded.
- Existing saves can use the toggles, but already generated planets are not retroactively regenerated.

# HardFog

HardFog 在 UXAssist 配置窗口中提供“黑雾操控”页，集中放置黑雾清理工具和已经合并进来的便利功能。偏作弊向的开关默认关闭，除非下面另有说明。

## 功能

- 清理当前星球的黑雾。
- 清理当前恒星系的太空黑雾。
- 为整个星系填满黑雾巢穴。
- 在当前星球生成黑雾星表废墟，并在空闲废墟上建造地热发电站。
- “黑雾降压器”：缓慢降低非主动攻击积累的巢穴威胁值，默认关闭。
- “中继站只发往信标”：黑雾只向 marker / 全息信标目标派出中继站，默认关闭。
- “研究倍速器”：把科技研究 hash 需求降低到约 1/36，默认关闭。
- “更好的矿物位置”：从每个星球的出生点开始，把未来生成的矿物重排到更容易规划的赤道经度窗口内，默认开启。
- “战斗机更远的飞行距离”：强化机甲战斗机的范围、伤害和生存能力，默认关闭。

## 设置

按 `Alt + 反引号` 打开 UXAssist 面板，在 HardFog 标签页中调整。

## 注意

- “更好的矿物位置”只影响之后生成的星球矿物。
- “战斗机更远的飞行距离”只影响机甲自己的战斗机，不影响战斗基地派出的战斗机。
- 已有存档可以使用这些开关，但已经生成过的星球不会被重新生成。
