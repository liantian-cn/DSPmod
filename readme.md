# Dyson Sphere Program Mods

这里收集了一组《戴森球计划》的小型功能 mod。它们彼此独立，可以按需要单独启用。

## Mod 列表

### FasterResearch

大幅减少科技研究需要的总 hash，让研究速度变快。默认效果约等于把研究需求降低到原来的 1/24，适合想减少等待时间、加快科技推进的存档。

### FogThreatDampener

缓慢降低太空黑雾巢穴的常态威胁值。它主要削弱戴森球等长期行为带来的威胁积累，不会直接清空主动攻击造成的威胁，也不会中断已经开始的太空突袭。

### OverpoweredMechaFighters

强化机甲自己的战斗机。机甲舰队战斗机会拥有更远的自动索敌范围、更大的交战范围、更高伤害，并变得不容易被击毁。该 mod 不影响战斗基地派出的战斗机。

### SmartRelayDispatch

让黑雾 Hive 只在有 marker / 全息信标目标时派出 Relay 中继站。没有指定目标时不会随机派出；有有效目标时会稳定派出，更适合玩家主动控制中继站落点。

### SurfaceRuins

在 UXAssist 配置窗口中加入“星表废墟”功能。可以在当前星球快速生成黑雾基地坑废墟，也可以在空闲废墟上直接补建地热发电站。

### VeinPlacement

优化星球矿物的位置分布。它不改变原版生成出的矿种、矿量和矿点数量，而是在生成后把矿组重排到更容易规划的中低纬度区域，并让同一矿簇更紧凑。

## 说明

这些 mod 通常没有游戏内配置界面，启用后即按各自默认逻辑生效。安装时把对应 mod 的 DLL 放入游戏的 BepInEx 插件目录即可。
