# Dyson Sphere Program Mods

这里收集了一组《戴森球计划》的小型功能 mod。它们大多彼此独立，可以按需要单独启用。

## Mod 列表

### HardFog

在 UXAssist 配置窗口中提供“黑雾操控”功能。可以清理当前星球的黑雾、清理当前恒星系的太空黑雾、为整个星系填满黑雾巢穴，也可以使用合并进来的星表废墟功能，在当前星球快速生成黑雾基地坑废墟或在空闲废墟上直接补建地热发电站。

HardFog 还内置“黑雾降压器”勾选项，默认关闭。启用后会缓慢降低太空黑雾巢穴的常态威胁值，主要削弱戴森球等长期行为带来的威胁积累，不会直接清空主动攻击造成的威胁，也不会中断已经开始的太空突袭。

HardFog 还内置“中继站只发往信标”勾选项，默认关闭。启用后让黑雾 Hive 只在有 marker / 全息信标目标时派出 Relay 中继站；没有指定目标时不会随机派出，有有效目标时仍按原版目标选择与落地检查流程稳定派出。

HardFog 还内置“研究倍速器”勾选项，默认关闭。启用后大幅减少科技研究需要的总 hash，默认效果约等于把研究需求降低到原来的 1/36，适合想减少等待时间、加快科技推进的存档。

HardFog 还内置“更好的矿物位置”勾选项，默认开启。启用后不改变原版生成出的矿种、矿量和矿点数量，而是在未来星球生成矿物后把矿组重排到更容易规划的中低纬度区域，并让同一矿簇更紧凑；已经生成过的星球和工厂不会被回溯调整。

HardFog 还内置“战斗机更远的飞行距离”勾选项，默认关闭。启用后只强化 `owner == -1` 的机甲舰队战斗机，让它们拥有 10x 自动索敌范围、10x 交战范围、10x 伤害，并设置为不容易被伤害路径击毁的 `isInvincible`；战斗基地派出的战斗机不受影响。

## 说明

这些 mod 通常没有游戏内配置界面，启用后即按各自默认逻辑生效；HardFog 的控制项位于 UXAssist 配置窗口。安装时把对应 mod 的 DLL 放入游戏的 BepInEx 插件目录即可。
