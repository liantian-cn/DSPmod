# HardFog 

Add some challenge and convenience to the Fog. Any cheating-oriented settings are turned off by default.

## Feature


* Clear the enemies on the current planet
* Clear the space enemies of the current star
* Fill the galaxy with Hive
* Reset the Hive in the current star

* The HIVE is sending out Relays more frequently
* Fog cannot steal electricity
* ray receiver receives at full power

* Hive in the current star get full energy and matter
* The Fog core on base get full energy and matter

* The HiveCore in the current star is invulnerable to attacks

## Configuration
Config panel is unified with UXAssist. Press `` Alt+`(BackQuote) `` to call up the config panel.


# HardFog 

为黑雾增加一些挑战性和便利性。任何作弊向的设置均为默认关闭。

## 功能


* 清理当前星球的黑雾
* 清理当前恒星的太空黑雾
* 为所有星系填满黑雾巢穴
* 为当前恒星重置黑雾巢穴

* 允许巢穴更频繁的发送的中继站
* 令黑雾无法偷电
* 射线接收器满速接收：即便戴森球未建造完。

* 玩家所在星系黑雾巢穴持续满能量
* 黑雾地面基地持续满能量

* 令当前恒星的黑雾巢穴核心无法被攻击


## 设置
